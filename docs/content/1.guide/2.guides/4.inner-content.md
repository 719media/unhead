---
title: Tag Inner Content
description: How tags are sorted and how to configure it.
---

To set the textContent of an element, use the children attribute:

```ts
useHead({
  style: [
    {
      children: `body {color: red}`,
    },
  ],
  noscript: [
    {
      children: `Javascript is required`,
    },
  ],
})
```
